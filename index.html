Mongo Appender for Log4Net

This project provides a Mongo based buffering appender for Log4net. The code is split up into a two main branche. One is for general Log4Net usage and the other is a Sitecore branch for use in that product. This is because the Sitecore product uses specific versions of the Mongo Driver and Log4Net.

The core code uses Nuget and will also be available as a NuGet package. The Sitecore related code is currently available as code only due to the dependencies. To build the Sitecore related code:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout sitecore-75

Longer term, the Sitecore branches will be replaced with assembly binding instructions and the core product will be available as a standalone Nuget package.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
