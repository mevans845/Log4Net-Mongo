Mongo Appender for Log4Net

This project provides a Mongo based buffering appender for Log4net. The core code uses Nuget and will also be available as a NuGet package. To build the bleeding edge code:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout develop

For use with Sitecore, assembly binding instructions will be provided.
